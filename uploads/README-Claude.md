# NordicRank Claude Design Brief

This is the main instruction file for Claude Design.

You are designing a new homepage for **NordicRank**, a premium Nordic backlink marketplace and link intelligence platform.

The goal is not to redesign a generic SEO agency website. The goal is to create a product-led SaaS/marketplace homepage that makes NordicRank feel like the smartest and most trustworthy way to find, buy and monitor backlinks across the Nordic markets.

## What To Do

Create a complete homepage design concept for NordicRank.

The design should include:

1. A full desktop homepage layout.
2. A responsive mobile direction.
3. A strong hero section with a clear product/platform signal.
4. A marketplace/product visual under or beside the hero.
5. Clear section structure for inventory, tools, guarantees, managed link building, agencies, publishers, founder credibility and FAQ.
6. Suggested copy for each section.
7. Notes on which reference screenshots influenced each section.
8. Placeholders for NordicRank brand assets, founder image and product animations/screenshots.

The homepage should feel premium, data-driven, Nordic, practical and conversion-focused.

## Business Context

NordicRank is a B2B link building marketplace and platform. It helps companies find, buy and monitor backlinks from vetted websites in Sweden, Norway, Finland and Denmark.

The platform is built for affiliates, e-commerce companies, local service businesses, SEO agencies, resellers, white-label agencies and site owners/publishers.

NordicRank works with demanding SEO customers and competitive verticals such as iGaming, finance and crypto.

The main value proposition:

> NordicRank helps agencies, affiliates and brands find, buy and monitor high-quality backlinks across the Nordic markets with vetted inventory, transparent SEO data, price guarantee, indexing guarantee, smart monitoring and managed link building support.

Core message:

### Smart Tools

NordicRank is not only a list of websites. It helps customers find the right backlinks.

#### Google SERP Scanner

A user enters a main keyword. NordicRank scans the top 100 Google results and identifies pages/sites already ranking for that keyword that are available in the inventory, including prices for buying guest posts or placements.

#### Semantic Link Matcher

A user enters a website or target URL. NordicRank analyzes and vectorizes the content, then matches it against inventory and top ranking pages to find the most thematically relevant link opportunities using cosine similarity.

#### Data-Driven Link Valuation

NordicRank uses data-driven SEO research, large-scale URL analysis and known link quality signals to estimate link value. The messaging should mention concepts such as:

- link equity
- PageRank-inspired thinking
- URL-level relevance
- site-level authority
- topical relevance
- link placement quality
- not just copying competitor backlinks blindly

Avoid risky wording around “Google API leak”. Do not frame it like a hack or secret exploit. Use careful wording:

> Built on data-driven SEO research, large-scale URL analysis and known link quality signals, so customers can prioritize links by expected value instead of gut feeling.

### Guarantees And Monitoring

NordicRank offers:

- Price guarantee.
- Indexing guarantee.
- Link monitoring.
- Smart alerts.
- Refund/replacement within 12 months when issues cannot be fixed.

Include details:

- If a link is found cheaper on another active established marketplace, NordicRank matches the price.
- Indexing is checked after 24 hours.
- It is checked daily until indexed.
- After indexation, it is checked monthly.
- NordicRank checks noindex, nofollow, anchor match, canonical and HTTP status.
- Customers get email alerts if something changes.
- Full refund if an article is not published, becomes noindex/nofollow and cannot be restored or fixed.
- Refund/replacement applies within 12 months.
