# Claude Design Iteration Prompt: Fix NordicRank Visual Direction

The previous output is not acceptable. It is too generic, flat and template-like. Redesign the homepage with a much stronger visual direction.

## Main Problem With Previous Draft

The previous draft looks like a generic SaaS template:

- Too much pale green/gray background.
- Too many basic rounded cards.
- Product mockups look like simple wireframes, not a premium platform.
- Layout is too boxy and repetitive.
- Typography is readable but not distinctive.
- Visual hierarchy is weak.
- The page does not feel like Linkbroker, PPC.io or ContextPR in quality.
- It does not feel premium enough for a serious Nordic link intelligence platform.

Do not continue in that style.

## New Visual Direction

Create a homepage that feels like:

> Linkbroker's marketplace/product structure + PPC.io's polished SaaS design quality + ContextPR's clear premium proof/process, but rebuilt as NordicRank's own platform.

The design should feel:

- Premium
- Confident
- Product-led
- Data-rich
- Nordic
- High-converting
- Built for serious SEOs/agencies
- More like a real product than a template

## Hard Design Constraints

### Layout

- The hero must feel more distinctive than a basic left-text/right-card SaaS layout.
- Include a strong product/platform surface in the first viewport.
- Use overlapping product UI panels, marketplace table previews, status widgets and metric cards.
- Create depth and visual tension, not just cards on a pale background.
- Avoid repetitive 3-card grids unless they are clearly differentiated.

### Product Mockup

The hero product mockup must look like a real NordicRank interface.

It should include:

- Inventory table
- Publisher/domain rows
- DR, RD, traffic, country, language, category and price
- Vetted status
- Google News classifier
- PBN risk
- Link monitoring status
- Indexed / Dofollow / Canonical OK / HTTP 200 checks
- SERP Scanner module
- Semantic Link Matcher module
- Smart alerts module

Make this product visual feel like the main asset of the homepage, similar in importance to Linkbroker's marketplace/product visuals.

### Color

Do not make the whole page pale mint/green.

Use a more premium palette:

- Mostly white/off-white surfaces
- Deep charcoal/navy text
- Controlled NordicRank green as accent only
- Subtle warm neutral or cool gray backgrounds
- Stronger contrast
- No muddy green-gray sections
- No overuse of green pills/icons/cards

### Typography

- Use stronger typographic hierarchy.
- Hero headline should feel premium and confident.
- Avoid generic uppercase green eyebrows everywhere.
- Make body copy readable and darker.
- Use tighter, more deliberate section headings.

### Components

Avoid basic generic cards.

Use richer components:

- Marketplace tables
- Split panels
- Dense metric rows
- Filter bars
- Status badges
- Comparison modules
- Workflow timelines
- Dashboard widgets
- Product UI cards with believable data

### Section Rhythm

The page should not be one card grid after another.

Use varied sections:

1. Immersive hero with marketplace/product visual.
2. Compact trust/data strip.
3. Large marketplace/inventory showcase.
4. Tool section with SERP Scanner and Semantic Matcher.
5. Guarantee/monitoring system visual.
6. Self-serve vs managed comparison.
7. Agency/API section.
8. Publisher section.
9. Founder credibility.
10. FAQ/final CTA.

Keep the core message:

H1:
Build better backlinks in the Nordics

Subcopy:
Find vetted Nordic websites, compare SEO metrics, buy guest posts and link inserts, and monitor every backlink after publication. NordicRank combines marketplace access, link intelligence and managed link building for Sweden, Norway, Finland and Denmark.

Hero CTAs:
- Browse inventory
- Book strategy call

Hero proof:
- Vetted Nordic inventory
- No dead zero-traffic sites
- Price guarantee
- Indexing guarantee
- Link monitoring included

## What To Avoid

Do not use:

- flat pale green page background everywhere
- generic SaaS illustration
- plain wireframe product cards
- repetitive cards with identical styling
- too many small green pills
- generic “boost your SEO” sections
- fake testimonials/logos
- huge empty whitespace with no product signal
- simplistic 3-step blocks that look like a template

## Required Output

Create a new design version, not a minor tweak.

Show:

1. Updated hero.
2. Updated product/platform mockup.
3. Updated section rhythm.
4. Stronger inventory/marketplace section.
5. Better tools section.
6. Better guarantee/monitoring section.
7. Better self-serve vs managed section.
8. Clear notes explaining what changed from the previous draft and which reference screenshots influenced each section.
