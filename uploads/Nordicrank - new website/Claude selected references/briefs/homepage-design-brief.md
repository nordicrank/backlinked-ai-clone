# NordicRank Homepage Design Brief

Datum: 2026-06-20
Syfte: Kort, praktisk designbrief för Claude Design. Den här filen destillerar innehållsdraften till designriktning, sektioner, visuella komponenter och prompt.

## Kort svar: vilken referens ska du utgå från?

Använd **Linkbroker som strukturreferens**, **PPC.io som känsloreferens** och **ContextPR som proof/process-referens**.

Men bygg inte en kopia av någon av dem.

## Rekommenderad blend

### Linkbroker: 50%
Använd för:
- marketplace-struktur
- publisher cards
- product sections
- tools + marketplace + managed service-logik
- tydlig funnel från self-serve till managed

Skippa:
- för mycket generisk backlink-marketplace-copy
- känslan av vanlig länkbutik
- för mycket pris-/paketfokus tidigt

### PPC.io: 30%
Använd för:
- modern AI-native SaaS-känsla
- stark produktconfidence
- mindre SEO-byrå, mer AI/data platform
- clean, confident, product-led hero

Skippa:
- om det blir för abstrakt
- om designen tappar inventory/marketplace-känslan
- för mycket startup-hype utan konkret data

### ContextPR: 20%
Använd för:
- processsektioner
- proof/case-känsla
- tydliga steg
- svensk premiumton
- trovärdighet runt Matyas/founder-story

Skippa:
- PR-byrå-känslan
- för mycket text utan produkt-UI

## Designpositionering

NordicRank ska kännas som:

**Premium Scandinavian SaaS + Nordic backlink marketplace + link intelligence platform**

Inte:
- klassisk SEO-byrå
- billig länkshop
- generisk AI-landingpage
- spreadsheet med filter
- casino-affiliate-tool

## Huvudbudskap

**Build better backlinks in the Nordics**

Subheadline:
NordicRank helps agencies, affiliates and brands find, buy and monitor high-quality backlinks across Sweden, Norway, Finland and Denmark with vetted inventory, transparent data, price guarantee and AI-driven link valuation.

## Hero ska sälja detta

1. Vetted Nordic inventory
2. Bättre länkval med data
3. Prisgaranti
4. Indexeringsgaranti/link monitoring
5. Marketplace + managed service i samma plattform

## Hero ska inte sälja allt

Lägg inte alla USP:ar i hero. Hero ska vara ren, stark och produktledd.

## Rekommenderad startsidesstruktur för design

Det fulla content-dokumentet har 24 sektioner. I designen ska det kännas som 10-12 stora block, annars blir sidan för tung.

### 1. Header
Nav:
- Marketplace
- SERP Scanner
- Semantic Matcher
- Monitoring
- Managed
- Agencies
- Pricing
- Login
- CTA: Book demo

### 2. Hero
Stor produktmockup + stark H1.

### 3. Proof bar
Kompakt rad:
- Vetted Nordic inventory
- No zero-traffic noise
- Price guarantee
- Indexing guarantee
- 7 days/week support

### 4. Problem
Most backlink marketplaces are full of noise.

Poänger:
- dead zero-traffic sites
- unclear price/data
- no monitoring after publication
- DR-only decisions
- too much manual work

### 5. Platform overview
Fyra cards:
- Find: marketplace, SERP Scanner, Semantic Matcher
- Compare: Ahrefs/DataForSEO/Majestic metrics
- Publish: guest posts, link inserts, content
- Monitor: indexing, nofollow/noindex/canonical/HTTP alerts

### 6. Marketplace preview
Stor dashboardsektion. Visa filter sidebar + publisher table/cards. Det här bör vara en av sidans största visuella showpieces.

### 7. SERP Scanner + Semantic Matcher
Två kolumner eller tabs.

SERP Scanner:
- keyword input
- Google top 100
- inventory match
- guest post price

Semantic Matcher:
- target URL input
- vector/cosine match
- relevance score
- article angle/entities

### 8. Link Monitoring
Dashboard/table med:
- live URL
- index status
- noindex/nofollow/canonical/HTTP
- anchor match
- last checked
- alert status

### 9. Managed Link Building + Hidden Inventory
Mörk/premium sektion eller tydlig bandsektion.
Sälj:
- budget planner
- competitor backlink import
- anchor distribution
- link velocity
- hidden inventory 800+
- outreach to sites that do not normally sell links

### 10. Agencies / reseller / API
B2B-section:
- reseller accounts
- discounts
- white-label link building
- white-label reports
- API

### 11. Quality / link valuation
Datadriven sektion:
- link equity
- PageRank-like signals
- source page quality
- anchor text/context
- entity relevance
- spam risk
- traffic/ranking history
- technical status

Gör detta som en visual Link Valuation Model, inte en lång textvägg.

### 12. Founder / proof
Matyas story kort:
- affiliate since 2011
- 7 figure exits
- built NordicRank because link planning needed better data
- two links from the same site can deliver very different value depending on structure, placement and target URL

### 13. Publishers CTA
Mindre sektion:
Are you a site owner?
- earn more from guest posts and banners
- recurring buyers
- keep control over topics

### 14. FAQ + final CTA
Kompakt FAQ.
Final CTA: Build better backlinks with better data.

## Visuell stil

### Layout
- Product-led SaaS.
- Dashboard previews, tables, cards, tabs.
- Sektioner ska vara scannable.
- Få stora visuella moments hellre än 24 jämnstora sektioner.

### Färg
Rekommendation:
- bas: near-black / white / light gray
- accent: emerald/green + blue/teal
- varnings/statusfärger: green for indexed, amber for attention, red for issue
- undvik lila/purple SaaS-gradient som känns generisk
- undvik beige/brown/cream
- undvik för mörk blå/slate över hela sidan

### Typografi
- Premium, tight, B2B SaaS.
- Inte marketing-hero med enorma ord överallt.
- Dashboard labels ska vara små och realistiska.

### UI-känsla
- Cards max 8px radius.
- Inga nested cards.
- Tydliga tabbar och tables.
- Använd status badges.
- Visa riktig produktdata, inte generiska lorem ipsum.

## Copy tone

- direct
- confident
- data-driven
- practical
- not hypey
