## Brand Positioning

NordicRank is a link building marketplace and link intelligence platform for the Nordic markets.

Position the company as:

> The smarter way to find, buy and monitor high-quality backlinks in the Nordics.

Core message:

NordicRank helps agencies, affiliates and brands find, buy and monitor high-quality backlinks across Sweden, Norway, Finland and Denmark with vetted inventory, transparent data, price guarantee, indexing guarantee and AI-driven link valuation.

## Homepage H1 Direction

Use this H1:

> Build better backlinks in the Nordics

Alternative if needed:

> The Nordic link building platform for smarter backlinks

Avoid overly long H1s like:

> Link building marketplace with biggest inventory for Nordic countries

That is too long and clunky.

## Suggested Page Structure

Create a full homepage design with these sections in this order:

### 1. Header / Navigation

Navigation items:

- Inventory
- Tools
- Managed Link Building
- Agencies
- Publishers
- Pricing / Book a call

Header CTA:

- Login
- Browse inventory / Get started

### 2. Hero

Include:

- H1
- Strong subcopy
- Two CTAs
- Proof chips
- Product/dashboard visual

The hero visual should show a believable NordicRank platform interface, for example:

- Inventory table with DR, RD, traffic, country, language, price, category
- Cards for vetted websites
- Link status checks: indexed, dofollow, canonical OK, HTTP 200
- SERP scanner / semantic matcher preview

### 3. Trust / Data Provider Strip

Show that NordicRank works with or enriches inventory using data from:

- Ahrefs
- DataForSEO
- Majestic
- Google SERP data
- Indexation checks

Do not overclaim official partnerships unless explicitly stated. Phrase as “data enriched with” or “metrics from”.

### 4. Vetted Nordic Inventory

Main message:

NordicRank does not flood users with dead domains. The marketplace is curated around real sites with organic traffic, relevance and useful metrics.

Include copy around:

- Sweden, Norway, Finland and Denmark
- Vetted sites with real traffic
- Exclusive direct agreements
- Hidden inventory available through managed link building
- No zero-traffic noise in the main marketplace
- Direct agreements with media companies, news sites, local newspapers and sports sites

Show a product UI inspired by Linkbroker marketplace cards, but make it NordicRank-specific.

### 5. Marketplace Filters / Metrics

Show filtering and comparison features:

- DR
- RD
- Organic traffic
- Price
- Category
- Country
- Language
- TLD
- Domain age
- Google News classifier
- Brand search
- Knowledge Graph
- Wikipedia page
- PBN risk
- Topical Trust Flow

Show that customers can see:

- Top ranking pages
- Ranking keywords
- Traffic history last 6 months
- Keyword distribution by position
- Example URLs from previous articles

### 6. Tools / Intelligence Section

Create a strong product-led section for smart link selection.

Include two main tools:

#### Google SERP Scanner

User enters a main keyword. NordicRank scans the top 100 Google results and identifies ranking pages/domains that are available in the inventory, including prices for guest posts or placements.

#### Semantic Link Matcher

User enters a website or target URL. NordicRank analyzes the content and matches it against inventory/top ranking pages to find thematically relevant sites using vector similarity/cosine similarity.

Also include:

- Data-driven link valuation
- Link equity
- PageRank-inspired thinking
- Not just copying competitor backlinks
- Decisions based on real URL-level and site-level signals

Be careful with “Google API leak” wording. Do not make it sound like a hack or compliance risk. Use this framing:

> Our link valuation is built on data-driven SEO research, large-scale URL analysis and known link quality signals, so you can prioritize links by expected value rather than gut feeling.

### 7. Guarantees / Monitoring

Show NordicRank guarantees clearly:

- Price guarantee: if a link is found cheaper on another active established marketplace, NordicRank matches it. Customer can email and get price matched without questions.
- Indexing guarantee: NordicRank checks indexation after 24 hours and keeps checking daily until indexed, then monthly.
- Link quality checks:
  - noindex
  - nofollow
  - anchor match
  - canonical
  - HTTP status
- Smart alerts by email if something changes.
- Refund/replacement within 12 months if an article is not published, becomes noindex/nofollow and cannot be fixed.

This should be a highly visible trust section.

### 8. Self-Serve vs Managed Link Building

Show two paths:

#### Self-serve marketplace

For users who want to browse, filter and buy links directly.

#### Managed link building

For customers who want NordicRank experts to handle the strategy, article creation, selection, outreach and publishing.

Managed link building includes:

- Expert link planning
- Free article creation
- Link suggestions based on budget
- Publishing workflow handled
- Hidden inventory with 800+ sites not listed publicly
- Link prospecting and outreach to site owners not actively selling links
- Competitor backlink import/review
- Budget planning
- White-label service for agencies

### 9. Agency / Reseller Section

Matyas Dios founded NordicRank after working with affiliate marketing since 2011, where backlinks were a major part of growth. He has built and sold sites across several verticals with seven-figure exits and built NordicRank to bring the metrics and workflow needed for serious link planning into one platform.

Two buyers can buy a link from the same site and get very different SEO value depending on how the link is structured, where it is placed and what page it points to. NordicRank helps users make those decisions with data.

## Tone

Use English for the page.

Tone should be:

- Clear
- Direct
- Premium
- Confident
- Data-driven
- Not fluffy
- Not hype-heavy

Avoid saying:

- “Cheapest in the industry”
- “Guaranteed rankings”
- “Google-safe” as an absolute claim
- “#1” unless backed by exact proof

Prefer:

- “Vetted”
- “Transparent”
- “Data-driven”
- “Monitored”
- “Nordic”
- “Relevant”
- “Real traffic”
- “Link equity”
- “Quality signals”
